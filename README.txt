INSTALL :

-Replace casco files in update\x64\dlcpacks\patchday4ng\dlc.rpf\x64\levels\gta5\vehicles.rpf\
-Replace handling file in update\x64\dlcpacks\mpheist\dlc.rpf\common\data 

-Addon: 
Drop mx5326 folder to update\x64\dlcpacks 
Edit dlclist.xml from update\update.rpf\common\data 
add this line 

<Item>dlcpacks:\mx5326\</Item>
 
save and replace it ,spawn name : mx5326
